# Public

This directory contains the public part of WatermelonKatana. It is the part that is accessible to the public. It contains the following subdirectories:

- `assets`: Contains all the assets used in the public part of WatermelonKatana.
- `css`: Contains all the CSS files used in the public part of WatermelonKatana.
- `js`: Contains all the JavaScript files used in the public part of WatermelonKatana.
- `html`: Contains all the HTML files used in the public part of WatermelonKatana.

The public part of WatermelonKatana is accessible to the public through the following URL: [https://watermelonkatana.com](https://watermelonkatana.com).
